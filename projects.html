Youtube Downloader

This is a simple but quite fast YouTube downloader. It is written in Python and uses the yt-dlp and PySide6 libraries.
It allows to download videos and playlists from an url or from the integrated web browser.
A.exeinstaller is available here and you can access the source code on Github.

Discord Bot

This is a Discord bot written in Python. It uses the discord.py library.
It is a bot that can play music from YouTube, has a few fun features and that can deliver some League of Legends statistics (currently only match history).
You can access the source code on Github or try it on my server.

WebApi

This is a WebApi written in Python using flask. It is not originally meant to be used by anyone but me, but you can use it or get ideas from it if you want.
Its main goal is to provide a simple yet powerful way to control some ws2812 LEDs connected to a Raspberry PI from the local network. Additionally, it offers access to the files stored on the PI and the camera if it is connected.
You can access the source code on Github.

Android Modulo calculator

This is a rudimentary modulo calculator for Android. It is written in Java and uses the Android SDK.
As for the WebApi, it is not originally meant to be used by anyone but me, so it lacks some features and is not very user-friendly. However, it might still be useful if you want to do some modulo calculations on the go.
You can download the apk file or access the source code on Github.
